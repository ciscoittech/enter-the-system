# Downloads — From Prompts to Pipelines

Downloadable files for each chapter. Each folder contains the **complete set of files** your system has after finishing that chapter.

## How to Use

1. Pick the chapter you're working on
2. Download or copy the files into your `my-ai-systems/` folder
3. Replace the `[BRACKETED PLACEHOLDERS]` with your real information
4. Follow the chapter instructions to test and customize

## Folder Guide

| Folder | Chapter | What's Inside |
|--------|---------|---------------|
| `before/` | Act 1 (Ch 1-3) | The vague one-liner prompts most people start with |
| `ch04-prompt/` | Ch 4: Structured Prompts | Root CLAUDE.md + 4 system CLAUDE.md files |
| `ch05-state/` | Ch 5: State Files | Everything from Ch 4 + 4 state files + updated CLAUDE.md files |
| `ch06-skill/` | Ch 6: Skills | Everything from Ch 5 + 5 skill files + updated CLAUDE.md files |
| `ch07-hook/` | Ch 7: Hooks | Everything from Ch 6 + 7 hook scripts + settings.json |

## The Progression

Each chapter builds on the last. The "finished" files from one chapter become the "starter" for the next:

```
before/          → vague prompts (no files, no system)
ch04-prompt/     → 5 CLAUDE.md files (root + 4 systems)
ch05-state/      → + 4 state files, CLAUDE.md files updated
ch06-skill/      → + 5 skill files in .claude/skills/
ch07-hook/       → + 7 hook scripts + settings.json in .claude/
```

## Important

- **Replace placeholders.** Every `[BRACKETED SECTION]` needs your real information.
- **These are starting points.** The chapter walks you through customizing each file.
- **Files evolve.** As you use your systems, you'll edit these files based on what works.
