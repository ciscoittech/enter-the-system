# Anti-Jargon Translation Table

## Hard Rules

If you can't translate it, cut it. If the reader needs a CS degree to parse it, rewrite it.

## When You MUST Use a Technical Term

Sometimes the UI uses a technical term (like "Global Instructions" in Claude's settings). In these cases:

1. Use the term exactly as it appears in the UI
2. Immediately follow with the plain-language translation in parentheses
3. Example: "Go to Settings > Cowork > Global Instructions (these are persistent notes Claude reads every time you start a new session)"
4. After the first use + translation, you can use the term without re-translating for the rest of the chapter
