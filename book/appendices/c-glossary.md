# Appendix C: Glossary

*Plain-language definitions for every technical term in this book. One sentence each.*

*To be populated as chapters are drafted — each new term gets added here.*

---

**Chat** — The regular conversational mode in Claude where you ask questions and get text answers back.

**Cowork** — The autonomous mode in Claude where you describe what you want done and Claude does the work on its own, producing finished files.

**Deliverable** — The finished file Claude creates for you (a spreadsheet, document, presentation, or organized folder).

**Global Instructions** — Persistent notes you write once that Claude reads every time you start a new Cowork session, so you don't have to repeat yourself.

**Permissions** — The rules about which folders Claude is allowed to open and what it can do (read, edit, or delete files).

**Prompt** — What you tell Claude to do — your instructions for the work you want done.

**Sub-task** — When Claude splits your request into smaller pieces of work and handles them separately (sometimes in parallel).
